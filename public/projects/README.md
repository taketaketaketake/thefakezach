## Image Specifications:
- **Format**: JPG or PNG
- **Size**: 400x400px (square aspect ratio)
- **File size**: Under 500KB for fast loading
- **Content**: Screenshots, logos, or representative images of each project

## Fallback:
If an image is missing, the system will automatically show the emoji instead.

## Tips:
- Use high-quality, clear images
- Make sure images look good when scaled down
- Consider adding your logo or branding to each image
- Screenshots of the actual projects work best
